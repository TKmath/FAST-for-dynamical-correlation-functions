# FAST-for-dynamical-correlation-functions

This repository contains numerical simulations for Fermionic-Adapted Shadow Tomography (FAST), as described in arXiv:2508.03192 (to be updated). We demonstrate the efficiency and physical applicability of the FAST method for the Su-Schrieffer-Heeger (SSH) model.

# Code Implementation
Parallel_comm_main.ipynb: The primary engine of the project. Implements numerical simulations of the SSH Model Hamiltonian, highlighting the advantage of the FAST over brute-force measurements. 

graph_data_.ipynb: Data visualization suite. Processes raw simulation output to generate the plots in our work.
